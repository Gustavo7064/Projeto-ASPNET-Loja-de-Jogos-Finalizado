<h1>🎮 RocketGames - Loja de Games em ASP.NET MVC</h1>

<p>
O <strong>RocketGames</strong> é uma aplicação web desenvolvida com <strong>ASP.NET Core MVC</strong>, simulando uma loja online de jogos digitais. 
O projeto foi criado com foco em desenvolvimento fullstack, integrando <strong>backend, frontend e banco de dados</strong> em uma aplicação completa.
</p>

<h2>🎯 Objetivo do Projeto</h2>
<p>
O objetivo deste projeto é aplicar na prática conceitos de desenvolvimento web, arquitetura MVC e integração com banco de dados, 
simulando um e-commerce real.
</p>

<ul>
  <li>Desenvolvimento de aplicações web com ASP.NET MVC</li>
  <li>Integração com banco de dados relacional</li>
  <li>Criação de sistema de autenticação</li>
  <li>Gerenciamento de produtos e usuários</li>
</ul>

<h2>⚙️ Funcionalidades</h2>
<ul>
  <li>Cadastro e login de usuários</li>
  <li>Listagem de jogos disponíveis</li>
  <li>Sistema de carrinho de compras (localStorage)</li>
  <li>Área de administração para gerenciamento de produtos</li>
  <li>Cadastro, edição e exclusão de jogos</li>
  <li>Busca e filtros de jogos</li>
  <li>Sistema de curtidas/avaliação</li>
</ul>

<h2>🏗️ Arquitetura do Projeto</h2>
<ul>
  <li><strong>Model:</strong> Representação das entidades (Jogos, Usuários, etc.)</li>
  <li><strong>View:</strong> Interface do usuário (Razor Pages + HTML/CSS)</li>
  <li><strong>Controller:</strong> Controle das requisições e regras de negócio</li>
</ul>

<h2>🧠 Conceitos Aplicados</h2>
<ul>
  <li>Padrão MVC (Model-View-Controller)</li>
  <li>CRUD completo (Create, Read, Update, Delete)</li>
  <li>Integração com banco de dados</li>
  <li>Autenticação e controle de acesso</li>
  <li>Manipulação de estado no frontend (localStorage)</li>
  <li>Separação de responsabilidades</li>
</ul>

<h2>💻 Tecnologias Utilizadas</h2>
<ul>
  <li>ASP.NET Core MVC</li>
  <li>C#</li>
  <li>MySQL</li>
  <li>HTML, CSS e JavaScript</li>
</ul>

<h2>🚀 Como Executar</h2>
<pre>
1. Clone o repositório
2. Configure a string de conexão com o banco de dados
3. Execute as migrations (se aplicável)
4. Rode o projeto via Visual Studio ou CLI (.NET)
</pre>

<h2>📌 Diferenciais</h2>
<p>
O projeto RocketGames se destaca por simular um fluxo completo de e-commerce, incluindo autenticação, gerenciamento de produtos e 
interação com o usuário, sendo uma base sólida para evolução para aplicações reais.
</p>
